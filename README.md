# node-dns-server

#### 介绍
node dns服务器
https加密连接阿里dns 223.5.5.5
防dns劫持

#### 使用说明

1.  克隆到本地
2.  yarn start 或 node dns-server-index.js 启动
3.  可使用pm2使任务在后台执行
